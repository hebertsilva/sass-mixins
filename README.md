SASS Mixins
===========

A collection of [SASS](http://sass-lang.com/ "SASS") mixins for your project.

* Animation 
   * Keyframes
* Background size
* Box shadow
* Opacity
* Clearfix
* Border radius
* Box-sizing
* Transition
* Columns
* Transform
    * Default
    * Translate
    * Skew
    * Scale
    * Rotate
* Gradients
    * Horizontal
    * Vertical
